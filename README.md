# MemRadar

RAM & SSD price tracker for PC builders.

Historical price charts, multi-retailer tracking, and price drop alerts — built for people who know exactly what they want and are waiting for the right price.

**memradar.com** (in development)

---

## Stack

- **Frontend:** Vanilla JS / HTML / CSS — GitHub Pages + Cloudflare
- **Backend:** Node.js — Vercel or Render
- **Database:** Supabase (Postgres)
- **Data sources:** Best Buy API (launch), Amazon (in progress)

## Status

Early development. Not yet live.
